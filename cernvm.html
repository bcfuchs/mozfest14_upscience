CernVM Web API

We'll show you how to use the CernVM WebAPI to build a fancy interface for controlling our Virtual Particle Collider.

We'll help you set up a local development environment and guide you on hooking up the Web API with Javascript and HTML5.

We'll also explain about the physics that is taking place.

And if we have enough time left, we might even connect the event information to the interfaces built by the Particle Clickers team
